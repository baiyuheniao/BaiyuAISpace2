# Skill 技能

Skill 把「一段常用的提示词 + 它需要的工具和知识」打包成可复用的技能，一次定义、随处使用。

## 技能的组成

每个 Skill 采用三段式定义：

| 字段 | 作用 |
|---|---|
| **名称** | 技能标识 |
| **描述** | 说明适用场景——开启「模型自主调用」时，模型靠它判断何时触发 |
| **指令** | 实际注入对话的提示词内容 |

此外还可以：

- **绑定 MCP 服务器**：激活技能时自动启用对应工具
- **绑定知识库**：激活技能时自动挂接检索
- **附加资源文件**：技能自带的参考材料

## 两种激活方式

1. **手动勾选**：在 Chat 输入区勾选要使用的技能
2. **模型自主调用**：开启后由 LLM 根据技能描述判断当前对话是否需要，自动按需触发

## 预设库

内置 **23 条预设技能**，覆盖开发者 / 商务 / 通用三类场景，在 **Skill（技能）** 页一键导入后即可使用，也可以在导入后按需修改。

## 与 Agent Team 联动

工作组中的每个 Agent 可以独立绑定技能，让不同 Agent 具备不同专长，见 [[Agent Team协作团队]]。
