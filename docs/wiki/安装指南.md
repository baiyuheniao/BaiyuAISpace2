# 安装指南

## 系统要求

| 平台 | 最低版本 |
|---|---|
| Windows | Windows 10 及以上 |
| macOS | 10.15 (Catalina) 及以上 |
| Linux | Ubuntu 20.04 及以上（或其他主流发行版） |

普通使用无需安装 Node.js / Rust 等开发环境，直接下载安装包即可。想从源码构建请看 [[开发者指南]]。

## 下载安装包

前往 [GitHub Releases](https://github.com/baiyuheniao/BaiyuAISpace2/releases) 页面，下载最新版本对应平台的安装包：

- **Windows**：NSIS 安装包（`.exe`），双击运行按提示安装
- 所有安装包均由 GitHub Actions 从公开源码自动构建，可溯源

## Windows SmartScreen 警告说明

Windows 安装包**暂未进行代码签名**。首次运行时 SmartScreen 可能弹出「已保护你的电脑」警告：

1. 点击 **「更多信息」**
2. 点击 **「仍要运行」**

这是独立开发者应用的常见情况，不代表安装包有安全问题——所有构建均由 GitHub Actions 从公开源码自动打包，构建过程公开可查。

## 自动更新

自 **v0.2.0-beta.3** 起，应用内置自动更新：

- 启动时后台检测 GitHub Releases 上的新版本
- 发现新版本后**左下角弹窗提示**，经你确认后才会下载安装
- NSIS 安装包原地升级，不会与旧版本共存；安装完成后自动重启应用

使用更早 Beta 版本（无自动更新）的用户，请手动到 [Releases](https://github.com/baiyuheniao/BaiyuAISpace2/releases) 下载新版覆盖安装。

## 数据存储位置（Windows）

应用的数据都保存在本地，不上传任何服务器：

| 内容 | 位置 |
|---|---|
| 运行日志 | `%APPDATA%\BaiyuAISpace2\logs\app_<日期>.log` |
| 会话 / 知识库等 SQLite 数据库 | `%APPDATA%\com.baiyu.aispace\` |
| API 密钥 | Windows 凭据管理器（Credential Manager），加密存储，不进数据库 |

macOS / Linux 对应使用系统 Keychain / Secret Service 存储密钥。

## 卸载

通过系统「应用和功能」正常卸载即可。如需彻底清除数据，可手动删除上表中的两个 `%APPDATA%` 目录，并在凭据管理器中删除相关条目。
