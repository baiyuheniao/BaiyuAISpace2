# 对话与模型接入

Chat 模块是应用的核心：统一接口对接 15+ 家 LLM 服务商与本地模型，并提供丰富的输入输出能力。

## 模型接入

- **多源统一接口**：OpenAI、Anthropic、Gemini、DeepSeek、Kimi 等 15+ 云端服务商 + 本地 Ollama / LM Studio + 任意 OpenAI 兼容自定义接口，完整清单见 [[支持的LLM提供商]]
- **会话内切换**：同一会话中可随时切换 API 配置，方便对比不同模型
- **自定义 Max Tokens**：每个 API 配置可单独设置最大输出 token

## 流式输出

- 实时打字机效果逐字呈现
- **真取消**：点击中断时后端通过 `CancellationToken` 真正终止请求，而非仅停止前端渲染
- 流式响应不设总超时，只用读间隔超时——长回答不会被中途掐断

## 多模态输入

| 类型 | 格式 | 支持范围 |
|---|---|---|
| 图片 | JPEG / PNG / GIF / WebP | 多数支持视觉的服务商 |
| 视频 | MP4 / WebM / MPEG | 仅 Gemini |

## 文档上下文注入

无需建知识库，直接把文档附加进对话，内容会解析后注入上下文：

- 支持 **PDF / Word / Excel / PPT / Markdown / 代码文件**
- 适合一次性问答场景；需要长期检索复用的资料请用 [[知识库RAG]]

## 思考模式（Extended Thinking）

输入区一键开关，按服务商自动适配参数：

- **Anthropic**：自适应 / 预算模式
- **Gemini**：`thinkingBudget`
- **SiliconFlow**：Qwen3 系列思考开关

## Prompt Caching

Anthropic 请求自动打 `cache_control` 断点，长对话可显著省钱提速，无需手动配置。

## 消息渲染

- **Markdown** 全量渲染，代码高亮（黑白 grayscale 主题，与整体设计一致）
- **Mermaid** 图表直接渲染
- **HTML 代码块**：沙箱 iframe 实时预览，可直接看模型生成的网页效果
- **XSS 防护**：所有渲染内容经 DOMPurify 净化

## 会话管理

- 会话自动永久保存到本地 SQLite
- **History（历史记录）** 页浏览、搜索、恢复历史会话
- 点击侧边栏 **New Session** 新建会话

## 与其他模块联动

在输入区可以直接挂接：

- **RAG**：勾选知识库，发送时自动检索相关片段注入上下文 → [[知识库RAG]]
- **MCP**：启用工具服务器，模型可自主发起多轮工具调用 → [[MCP工具]]
- **Skill**：手动勾选技能，或开启「模型自主调用」 → [[Skill技能]]
