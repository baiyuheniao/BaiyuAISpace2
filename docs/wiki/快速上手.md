# 快速上手

本页带你在 5 分钟内完成从零到第一次对话。前提：已按 [[安装指南]] 装好应用。

## 第一步：配置 LLM API

1. 打开应用，在左侧边栏进入 **Settings（设置）**
2. 在 **LLM API 配置** 区域点击新建，填写：
   - **服务商**：从 15+ 家提供商中选择（OpenAI、Anthropic、Gemini、DeepSeek、Kimi、智谱、通义……完整列表见 [[支持的LLM提供商]]），或选「自定义」填任意 OpenAI 兼容接口的 Base URL
   - **API Key**：从服务商控制台获取；密钥会加密存入系统密钥链，不会明文落盘
   - **模型**：从预置列表选择，也可以直接输入模型名称添加新模型（各家模型更新频繁，以官方文档为准）
   - **Max Tokens**（可选）：每个配置可单独设置最大输出 token
3. 保存并**激活**该配置

> 💡 没有云端 API Key？可以用本地模型：先到 **Local（本地部署）** 页安装并启动 Ollama，下载一个模型，再回设置页添加 Ollama 类型的配置。详见 [[本地部署]]。

## 第二步：开始对话

1. 侧边栏进入 **Chat（对话）**，或点击 **New Session** 新建会话
2. 在输入框输入内容发送，回复以流式打字机效果实时呈现，随时可点击中断
3. 会话自动永久保存，可在 **History（历史记录）** 中随时回看；会话内也可以随时切换 API 配置

输入区功能一览（详见 [[对话与模型接入]]）：

- **附件**：发送图片（多数服务商支持）或视频（仅 Gemini）
- **文档**：把 PDF / Word / Excel / PPT / Markdown / 代码文件直接注入对话上下文，无需建知识库
- **思考模式**：一键开关 Extended Thinking（支持 Anthropic / Gemini / SiliconFlow Qwen3 等）
- **RAG / MCP / Skill**：挂接知识库、工具服务器和技能

## 三类 API 配置的区别

设置页可以配置三类 API，各司其职：

| 类型 | 用途 | 什么时候需要 |
|---|---|---|
| **LLM API** | 对话生成 | 必须，至少一个 |
| **Embedding API** | 知识库文档向量化 | 使用 RAG 知识库时 |
| **Reranker API** | 检索结果精排（Cohere 兼容） | 可选，想提升 RAG 检索质量时 |
